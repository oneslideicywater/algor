== 快速幂

求a^b,需要将b转换为二进制。比如b=10

利用a^(b+c) = a^b * a^c,得到：

image::img/000002.png[]

可以发现项前面的系数就是1010的倒置。

[source,java]
----
   static int Pow(int a,int b){
        int ans = 1;
        int base = a;
        while(b!=0){
            if((b & 1)==1) ans *= base;
            base *= base;
            b >>= 1;
        }
        return ans;
    }
----

这样的话，我们把CPU做乘法的次数从10次，变成4次。
